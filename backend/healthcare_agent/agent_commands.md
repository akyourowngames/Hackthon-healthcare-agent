# Agent Commands

These phrases are the terminal agent command surface. Edit this file to tune
how the agent understands operator instructions without changing code.

## Commands

- process_input: process | ingest | process input | scan input | ingest input | read input | process reports | ingest reports | read documents | process documents
- list_reports: list | reports | list reports | show reports
- status: status | agent status
- help: help | commands
- exit: exit | quit | bye
- show: show
- search: search
- ask: ask
