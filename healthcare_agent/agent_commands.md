# Agent Commands

These phrases are the terminal agent command surface. Edit this file to tune
how the agent understands operator instructions without changing code.

- list_reports: list | reports | list reports | show reports
- dedupe: dedupe | deduplicate reports | clean duplicate reports
- report_inventory: which report you have access to | which reports you have access to | what reports do you have | what reports are in memory | show stored reports | reports in memory
- latest_patient: whats the patient name | what's the patient name | what is the patient name | patient name | who is the patient
- latest_report_name: whats the report name | what's the report name | what is the report name | report name | latest report name
- extraction_help: how can i make him extract the details | how can i extract the details | how to extract report details | extract the details | upload pdf | upload report | add report
- quick_reply: hi | hello | hey
- status: status | agent status
- help: help | commands
- exit: exit | quit | bye
- show: show
- search: search
- ask: ask
